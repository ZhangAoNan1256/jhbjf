## 🎯 技术实现特点

### 1. 零依赖架构
- ✅ 无需Node.js环境
- ✅ 无需npm包管理
- ✅ 无需Webpack/Vite打包
- ✅ 双击HTML直接运行

### 2. 技术栈极简
```html
<!-- 仅引入3个CDN资源 -->
Bootstrap 5 CSS  → UI样式
Bootstrap 5 JS   → 交互组件
Axios            → HTTP请求
```

### 3. 状态共享机制
```javascript
// 使用localStorage在所有页面间共享userId
localStorage.setItem("userId", userId);  // 存储
localStorage.getItem("userId");          // 读取
```

### 4. 统一错误处理
```javascript
// global.js中定义的统一错误处理函数
function handleError(error) {
    if (error.response) {
        tip(`错误: ${error.response.data?.message}`);
    } else {
        tip('网络请求失败，请检查后端服务是否启动');
    }
}
```

## ⚡ 性能优化

1. **CDN加载**：所有第三方库使用CDN，利用浏览器缓存
2. **按需加载**：每个页面独立，不加载无关资源
3. **并行请求**：point.html使用Promise.all同时请求余额和日志
4. **本地存储**：userId使用localStorage，减少重复输入

## 📝 代码规范

### 1. 命名规范

- ID命名：小写+下划线（userTableBody, trafficForm）
- 类名：kebab-case（user-row, coupon-card）

### 2. 注释规范
```javascript
// 单行注释：功能说明
/**
 * 多行注释：复杂逻辑说明
 */
```

### 3. 异步处理
统一使用async/await语法：
```javascript
async function loadUsers() {
    try {
        const response = await axios.get('/api/user/list');
        displayUsers(response.data);
    } catch (error) {
        handleError(error);
    }
}
```
